# Proyecto Final Programa Oracle One y Alura LATAM

Este proyecto está obteniendo datos de una API falsa implementada en Vercel:[
https://fake-api-aluraflix-two.vercel.app/)
Este proyecto representa el desafío React, que consiste en crear una aplicación web similar a Netflix. El diseño del proyecto se basa en el diseño Figma (https://www.figma.com/file/jWsqqszcgLAecpsJArCVgI/AluraFlix-ESP-(Copy)?type=design&mode=design&t=SeuJUCGHT8jWQiZf-1) proporcionado por Alura, pero usted tenga la flexibilidad de personalizar el diseño según sus preferencias.
